##  What is Pandas?

> Pandas is used for **data manipulation and analysis**.
> It works with structured data like **Excel sheets and CSV files**.

```
NumPy  → Works with numbers & arrays 
Pandas → Works with tables & datasets 
```

## Pandas Series

A Series is a **1D labeled array** — like a single column in Excel.

```python
import pandas as pd

# Create Series
s = pd.Series([10, 20, 30, 40, 50],
               index=['a', 'b', 'c', 'd', 'e'])

print(s)
# a    10
# b    20
# c    30
# d    40
# e    50

print(s['b'])    # 20
print(s.mean())  # 30.0
```

##  Pandas DataFrame

A DataFrame is a **2D table** — like a full Excel sheet.

### Create DataFrame
```python
# From dictionary
data = {
    'Name':   ['Alice', 'Bob', 'Charlie', 'Diana'],
    'Age':    [25, 30, 35, 28],
    'Salary': [50000, 60000, 75000, 55000],
    'City':   ['Mumbai', 'Delhi', 'Pune', 'Chennai']
}

df = pd.DataFrame(data)
print(df)

#       Name  Age  Salary     City
# 0    Alice   25   50000   Mumbai
# 1      Bob   30   60000    Delhi
# 2  Charlie   35   75000     Pune
# 3    Diana   28   55000  Chennai
```

###  Access Columns & Rows
```python
# Single column
print(df['Name'])

# Multiple columns
print(df[['Name', 'Salary']])

# Single row by index
print(df.iloc[0])

# Row by label
print(df.loc[0])

# Specific cell
print(df.loc[0, 'Name'])    # Alice
print(df.iloc[0, 1])        # 25
```

## Grouping Data

```python
# Group by City and get mean salary
city_salary = df.groupby('City')['Salary'].mean()
print(city_salary)

# Group by City and count employees
city_count = df.groupby('City')['Name'].count()
print(city_count)

# Multiple aggregations
summary = df.groupby('City').agg({
    'Salary': ['mean', 'max', 'min'],
    'Age': 'mean'
})
print(summary)
```

## Useful DataFrame Operations

```python
# Add new column
df['Tax'] = df['Salary'] * 0.1

# Rename columns
df.rename(columns={'Name': 'Employee'}, inplace=True)

# Drop column
df.drop('Tax', axis=1, inplace=True)

# Sort by salary
df.sort_values('Salary', ascending=False, inplace=True)

# Reset index
df.reset_index(drop=True, inplace=True)

# Drop duplicates
df.drop_duplicates(inplace=True)

# Value counts
print(df['City'].value_counts())
```

## NumPy + Pandas Together

```python
import numpy as np
import pandas as pd

# Create DataFrame
df = pd.DataFrame({
    'Score': [85, 92, 78, 95, 88, 72, 90]
})

# Use NumPy with Pandas
df['Score_Scaled'] = (df['Score'] - np.mean(df['Score'])) / np.std(df['Score'])
df['Score_Squared'] = np.square(df['Score'])
df['Grade'] = np.where(df['Score'] >= 85, 'A', 'B')

print(df)
```
