##  1. Definition

>Machine Learning is the branch of Artificial Intelligence that aims to make computer or machine to
learn from data and make actions and predication without explicitly programmed for each task.

> **Simply put:** Instead of writing rules manually, you give data to the machine to learn automatically.

##  2. How Machine Learning Works

>> DATA → ALGORITHM → MODEL → PREDICTIONS

->Step by Step Process:
1. **Collect Data** — Gather relevant dataset
2. **Preprocess Data** — Clean and prepare data
3. **Choose Algorithm** — Select the right ML algorithm
4. **Train Model** — Feed data to the algorithm
5. **Evaluate Model** — Test how well it performs
6. **Deploy Model** — Use it in real world


## 3. Types of Machine Learning

> 3.1 SUPERVISED LEARNNG:-

- Model learns from **labeled data**
- Input and output are both provided
```python

# Example: Supervised Learning
X = [[1], [2], [3], [4]]   # Input (features)
y = [2, 4, 6, 8]            # Output (labels)
# Model learns: y = 2 * X

> 3.2 UNSUPERVISED LEARNING:-

- Model learns from **unlabeled data**
- Finds hidden patterns by itself
- **Examples:** Customer segmentation, Anomaly detection

# Example: Unsupervised Learning
data = [[1,2], [1,4], [5,8], [6,9]]
# Model groups similar data points together
# No labels provided!

# SEMI SUPERVISED(train by itself)

> 3.3 REINFORCEMENT LEARNING:-
- Model learns by **trial and error**
- Gets rewards for correct actions
# Examples:** Chess AI, Self driving cars, Game playing bots,Recommendation system
